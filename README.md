# Invitación Revelación de Sexo 👶👧

Proyecto de invitación web premium para evento de revelación de sexo, con cuenta regresiva, efectos de glassmorphism y registro de asistencia (RSVP) conectado a Google Sheets.

## Características
- Diseño Moderno y Elegante.
- Fondo de video inmersivo.
- Registro de asistencia en tiempo real.
- Conexión con Google Sheets para gestión de invitados.

## Despliegue
Este proyecto está listo para ser desplegado en **Easypanel** usando el Dockerfile incluido.
